# Kemi Project

## Overview
This project is designed to provide an educational platform for learning about various aspects of chemistry. It includes interactive quizzes and detailed information on key topics within the field.

## Project Structure
The project consists of the following files and directories:

- **Style och js/**
  - `style.css`: Contains the CSS styles for the project, defining the visual appearance of the HTML pages.
  - `Slutproject.js`: Contains JavaScript code that may include functionality for interactivity or dynamic content on the web pages.

- **sidor/**
  - `Index.html`: Serves as the main homepage, featuring three options representing the subgroups of chemistry: Atoms and the periodic table, chemical bonds, and acids and bases. Each option includes a brief introduction and links to their respective pages.
  - `Atomer+periodiska.html`: Contains information about atoms and the periodic table, organized with images and examples, similar to a typical chemistry textbook.
  - `kemiskaBindningar.html`: Provides information about chemical bonds, structured with images and examples for better understanding.
  - `syror+Baser.html`: Includes content about acids and bases, presented with images and examples, akin to a chemistry textbook.
  - `AtomQuiz.html`: Contains a quiz related to atoms and the periodic table.
  - `bindningarQuiz.html`: Contains a quiz related to chemical bonds.
  - `SyraBasQuiz.html`: Contains a quiz related to acids and bases.

## Usage
To use this project, open the `Index.html` file in a web browser. From there, you can navigate to the different sections of chemistry by clicking on the provided links. Each section will offer detailed information and quizzes to test your knowledge.
